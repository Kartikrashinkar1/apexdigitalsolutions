Core Web Vitals are a set of performance metrics introduced by Google to measure user experience. They focus on three main aspects:

1. Largest Contentful Paint (LCP)

LCP measures how fast the main content of your page loads. A good LCP is under 2.5 seconds.

2. First Input Delay (FID)

FID checks how quickly your website responds when users interact (click, tap, etc.). The ideal score is below 100ms.

3. Cumulative Layout Shift (CLS)

CLS measures how stable your content is while loading. Pages should stay visually stable, with a score of less than 0.1.
